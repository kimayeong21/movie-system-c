영화 추천 프로그램 - 코드 설명

이 프로그램은 C 언어로 작성된 영화 추천 시스템입니다. 사용자 로그인, 회원가입, 영화 정보 관리, 키워드 검색, 영화 추가, 수정 및 삭제 기능을 제공합니다.

기본 구조

프로그램은 주요 기능을 함수로 나누어 구현했습니다. 주요 함수 목록은 다음과 같습니다:

  • show_main_menu() - 메인 메뉴를 출력합니다.
  • login() - 사용자 로그인 기능입니다.
  • signup() - 회원가입 기능입니다.
  • admin_mode() - 관리자 모드로 전환합니다.
  • show_movie_list() - 영화 목록을 출력합니다.
  • search_movie_by_keyword() - 키워드를 사용한 영화 검색 기능입니다.
  • movie_detail() - 영화 상세 정보를 출력합니다.
  • admin_menu() - 관리자 메뉴를 출력합니다.
  • add_movie() - 영화를 추가합니다.
  • modify_movie() - 영화를 수정합니다.
  • delete_movie() - 영화를 삭제합니다.
  • view_logs() - 관리자 로그를 봅니다.
  • login_failed() - 로그인 실패 시 처리합니다.

데이터 파일 구조

프로그램은 CSV 형식의 데이터 파일을 사용하여 사용자 정보와 영화 정보를 저장합니다.

1. 사용자 정보 파일 (user.csv)

사용자 아이디와 비밀번호를 저장합니다.

아이디,비밀번호
penguin,pass123
eagle,pass456

2. 영화 정보 파일 (movie.csv)

영화 ID, 제목, 장르, 개봉 연도, 평점 정보를 저장합니다.

영화ID,영화 제목,장르,개봉 연도,평점
1,기생충,드라마,2019,8.6
2,괴물,스릴러,2006,7.9
3,암살,액션,2015,7.3

3. 추천 정보 파일 (recommendation.csv)

추천받은 영화 정보와 추천 일자를 저장합니다.

아이디,추천 날짜,추천 영화 제목,개봉 연도,평점
penguin,2024-09-10,기생충,2019,8.6
eagle,2024-09-11,암살,2015,7.9

4. 관리자 작업 로그 파일 (admin_log.csv)

관리자의 작업 기록을 저장합니다. 작업 ID, 날짜, 작업 종류, 영화 제목이 포함됩니다.

관리자 아이디,작업 날짜,작업 종류,영화 제목
admin123,2024-09-10,추가,기생충
admin123,2024-09-11,수정,암살
admin123,2024-09-12,삭제,괴물

관리자 기능

관리자 모드에서 영화 추가, 수정, 삭제 및 로그 기록 보기 기능을 사용할 수 있습니다.

코드 예제

아래는 show_main_menu() 함수 예제입니다:

void show_main_menu() {
    printf("==========================================\\n");
    printf("          🎬 영화 추천 프로그램 🎬\\n");
    printf("==========================================\\n");
    printf("[1] 로그인\\n");
    printf("[2] 회원가입\\n");
    printf("[3] 관리자 모드\\n");
    printf("[4] 종료\\n");
    printf("------------------------------------------\\n");
    printf("번호를 입력하여 선택하세요: ");
}

오류 처리

잘못된 입력에 대한 오류 처리 및 파일 불러오기/저장 실패 시 오류 메시지를 표시합니다.

보안

프로그램 실행 시 #define _CRT_SECURE_NO_WARNINGS를 추가하여 MSVC의 경고를 억제하고 있습니다.
