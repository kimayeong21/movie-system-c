## 📌 프로젝트 개요
사용자의 선호도(장르, 평점 등)를 기반으로 맞춤형 영화 추천을 제공하는 프로그램입니다.  
다양한 영화 정보를 효율적으로 탐색할 수 있도록 설계되었습니다.

## 🚀 주요 기능
- 회원가입 / 로그인 기능
- 영화 검색 기능
- 맞춤형 영화 추천
- 영화 평점 입력 (1.0 ~ 5.0)
- 관리자 기능 (영화 추가 / 수정 / 삭제)

## 🛠 개발 환경
- 개발 언어: C
- 데이터 저장 방식: 파일 시스템 (TXT)
